This project is a web application built using Flask, designed to provide real-time system and process monitoring through RESTful APIs. The application leverages the psutil library to gather system metrics and process information, which can be accessed via API endpoints.

Key Features
  1. Real-Time Process Information:
    • Description: The application provides an API endpoint to retrieve real-time information about running processes on the system.
    • Endpoint: /api/process
    • Example: A client can send a GET request to /api/process to receive a JSON response containing details such as:
      • Process ID
      • Name
      • Memory usage
      • Number of threads
      • CPU times for each running process
  2. Real-Time System Metrics:
    • Description: The application provides an API endpoint to retrieve real-time system metrics, including CPU usage, memory usage, disk usage, network I/O, and disk I/O.
    • Endpoint: /api/monitor
    • Example: A client can send a GET request to /api/monitor to receive a JSON response containing the following system metrics:
      • Current CPU usage percentage
      • Memory usage percentage
      • Disk usage percentage
      • Network bytes sent/received
      • Disk bytes read/written
